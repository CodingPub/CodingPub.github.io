更换收藏夹图标

这个跟主题其实没有关系，只是因为属于样式的自定义，所以放在这里说明。在 hexo/source 目录下放上你的收藏夹图标，命名为favicon.ico即可，建议尺寸为 32*32。

还可以在这里添加苹果设备图标，命名为apple-touch-icon.png，建议尺寸为 120*120。

显示文章目录

在文章的 Front-matter 中添加toc: true就可以让该篇文章显示目录，可以修改 hexo/scaffolds/post.md 模板文件，这样之后创建的文章默认都显示文章目录。

支持Google搜索

  • 添加站点地图插件

    修改 hexo 配置

    重新生成文件后访问 http://localhost:4000/sitemap.xml

  • 向 Google 提交站点地图

    1. 在 Google 的网站管理员中心点击“添加属性”，输入你的站点，并接续。
    2. 根据提示验证网站的所有权，注意此处如果选择上传一个html文件进行验证操作，不要通过 hexo 上传，hexo 会修改 Google 指定的文件，导致验证失败。
    3. 在Search Console页面找到刚添加的网站
    4. 上传站点地图
    5. 等待 Google 审核，大概1天到3天~，审核通过之前，先把搜索框隐藏吧。

支持百度搜索

  • 添加站点底图插件

    修改 hexo 配置

  • 向百度提交站点底图

    1. 在百度站点管理页面添加站点，并根据提示完成验证操作。
    2. 在百度链接提交页面，找到自动提交 - sitemap入口，填写 baidusitemap.xml 文件的地址即可。
    3. 等待百度审核、收录。

Google Analytics

  1. 使用 Google 帐号登录 Google Analytics
  2. 根据提示进行注册操作，完成注册后通过 管理 - 选择用户 - 选择媒体 - 跟踪信息 - 跟踪代码 可以查到跟踪 ID
  3. 修改主题配置，设置 google_analytics

百度 Analytics

  1. 登录或注册百度统计

  2. 根据提示创建网站，获取百度 Analytics ID

  3. 修改主题配置，设置 baidu_analytics

  4. 重新发布博客

  5. 验证统计正常安装

支持 Disqus 评论

对多说和 Disqus 做了一些对比，国内而言，多说会快一些，不过产品似乎不太稳定，偶尔会抽风，目前的维护力度似乎也不到位， Disqus 速度会慢些，有被墙的风险，使用门槛相对也比较高。虽然 Disqus 门槛高些，最终还是决定试用看看。

  1. 注册帐号 Disqus 帐号，可以如果有 Gmail 的话，可以快速登录
  2. 验证邮箱后完善个人信息
  3. 在 Setting - Add Disqus To Site 入口进入配置页面，根据引导创建 Choose your unique Disqus URL

  4. 修改主题配置，开启 Disqus 评论功能，填入第 3 步创建的 shortname 即可

  5. 如果需要关闭部分文章的评论功能，可以在文章或 page 的 Front-matter，添加配置
