创建个人博客（一）

Mar 29, 2016 | 其他 | Hits

毕业也有三年多了，受到工作环境的影响，差不多一年多前才开始接触Github，开始在上面积累自己感兴趣的源码，一段时间过去，收获是有的，但总觉得缺少一点沉淀，于是想开始写些博客。

所谓工欲善其事必先利其器，在接触新事物之初，先做一定的调查，找出自己喜欢的产品，能在一定程度上避免走弯路，个人感觉这个过程还是很有必要的。正式开始写博客之前，了解了一些搭建博客的工具，比如 WordPress、Github Pages、Jekyll、Hexo 等等。过程就不一一表述了，作为一名有（xiang）上（yao）进（zhuang）心（bi）的程序员，搭建一个私人博客那是必须的。

最终选定了 Github Pages + Hexo，至于选择这两个工具的理由，免费是第一位的，简单、易用、易部署、支持换主题、支持自定义、支持插件、支持 Markdown 都是加分项。

虽说其他产品有稍微了解了一下，但毕竟没有用过，也就不多做评价，介绍下 Github Pages 和 Hexo 。
